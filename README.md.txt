# Multilingual Sign Board Translation System

## Project Overview

The Multilingual Sign Board Translation System is designed to help users understand sign boards written in different languages. The application extracts text from sign board images using Optical Character Recognition (OCR) and translates the detected text into the user's preferred language.

## Features

* Extracts text from sign board images
* Supports multiple language translations
* OCR-based text recognition
* User-friendly interface
* Helps overcome language barriers

## Technologies Used

* Python
* OpenCV
* EasyOCR / Tesseract OCR
* Translation API
* NumPy

## How It Works

1. Upload a sign board image.
2. OCR extracts the text from the image.
3. The extracted text is processed.
4. The text is translated into the selected language.
5. The translated result is displayed to the user.

## Applications

* Tourism and travel assistance
* Educational purposes
* Multilingual communication
* Accessibility enhancement

## Future Enhancements

* Real-time camera translation
* Voice output for translated text
* Support for additional languages
* Improved OCR accuracy
